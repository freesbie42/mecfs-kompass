MECFS Kompass
===================================

Das Projekt **MECFS Kompass** soll eine übersichtliche, aktuelle, einfache und vorallem 
energiesparende Handreichung für die Patient-Arzt-Kommunikation sein.
Vom ersten Verdacht, über das erste Arztgespräch über die folgenden Facharzttermine.

Ein erster Anlaufpunkt in diesem Projekt sollte die Seite :doc:`fahrplan` sein. 


.. note::

   Dieses Projekt ist unter ständiger Entwicklung
